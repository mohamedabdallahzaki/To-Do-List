# To-Do-List


A basic Todo List web application built with ASP.NET Core 8 (MVC/Razor Pages).  
The app allows users to **add**, **view**, and **delete** todo items using in-memory storage .

---

##  Features

- View all todo items on the home page.  
- Add new todo items using a simple form.  
- Delete items with a single click.  
- (Optional Bonus) Mark items as completed using a checkbox.
